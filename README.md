# go_vue_todo
Go, Vue.js, MySQLによるToDoアプリ

# 使用技術
Docker環境上で以下3つに対応したコンテナを用意し、docker-composeでスタック
- フロントエンド：Vuetify（Vue.js）
- バックエンド：Gin（Go）
- データベース：MySQL
